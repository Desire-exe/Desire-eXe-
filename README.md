## 🛠️ **Installation**

### Prerequisites
- Get your Gemini API key from [Google AI Studio](https://aistudio.google.com)
- Configure your settings in `config.json`

### Windows
```bash
git clone https://github.com/Desire-exe/Desire-eXe-
cd whatsapp-bot
npm start
```

### Linux
```bash
sudo apt update
sudo apt install nodejs npm
git clone https://github.com/Desire-exe/Desire-eXe-
cd whatsapp-bot
npm start
```

### Termux
